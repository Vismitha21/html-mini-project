# html-mini-project
Study Habit Analyser

A beginner-friendly multi-page HTML project designed to help students track and understand their study habits, productivity, and daily routines.

Project Description

The Study Habit Analyser is a simple website created using only HTML.
It allows students to enter their study-related details, understand healthy study habits, and explore productivity tips.

This project focuses on practicing important HTML concepts through a real-world student-based idea.

Features

- Multi-page website
- Navigation between pages
- Student study habit form
- Productivity and lifestyle inputs
- Study tips section
- Study habits table
- Reset and submit options
- Clean and structured webpage layout

Files Included

- "index.html" – Home page
- "form.html" – Study habit form page
- "tips.html" – Study tips and table page
